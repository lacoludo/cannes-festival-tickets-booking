### Prerequisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/) (Optional)

### Installing dependencies

with npm (installed with Node.js)

```
> npm install
```

or with Yarn

```
> yarn install
```

### Running the server

with npm (installed with Node.js)

```
> npm start
```

or with Yarn

```
> yarn start
```
